1)  Ekstrak File ini

2) Copy dan paste folder 'baresbin' ke folder htdocs XAMPP

3) Buka XAMPP lalu jalankan Apache dan MySQL

4) Buka localhost/phpmyadmin

5) Buat database baru dan beri nama 'baresbin' (tanpa tanda petik)

6) Buka CMD/Terminal dan masuk ke folder 'baresbin' tadi.

7) Jalankan perintah "php artisan migrate" untuk membuat tabel yang dibutuhkan.

8) Lalu menjalankan Seeder untuk membuat akun admin dengan perintah:
     "php artisan db::seed --class=CreateUsersSeeder"
Note : Akun admin default pada seeder adalah ( 'email' =>'admin@baresbin.com', 'password' => 'baresbin').
Kamu bisa memodifikasi data akun dengan mengubah file seeder di folder database\CreateUsersSeeder.php

9) ketikkan perintah 'php artisan serve'

10) Buka 'localhost:8000 'di browser
